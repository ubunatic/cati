terminal image viewer

cat for
images.

cati renders PNG and JPEG files in your terminal using Unicode half-block characters and 24-bit ANSI true-color — no external dependencies, no image protocol required.

Get started View source
cati — bash
$ cati --play --fps 2 assets/
capabilities

Everything you need to view images in the terminal

No framebuffer, no image protocol, no root access.

▀▄

Half-block rendering

Each terminal cell encodes two vertical pixels using ▀ ▄ █ and 24-bit true-color ANSI, doubling effective vertical resolution.

📁

Folder support

Pass a directory to render all images inside, sorted. Use -r to recurse into subdirectories automatically.

▶

Animation playback

--play loops through frames at --fps frames per second. Pre-loads everything for smooth, tear-free rendering.

⟷

Auto-scaling

Detects terminal width via TIOCGWINSZ and scales images down to fit — images never overflow or break your layout.

🖼

PNG & JPEG

Supports PNG (including full transparency) and JPEG out of the box. More formats can be added by registering Go image decoders.

⌨

Single-key quit

Press q, ESC, or Ctrl+C to stop playback instantly. Terminal state is always restored cleanly.

technique

Two pixels, one cell.

Unicode block characters divide a terminal cell into a top half and a bottom half. Setting foreground and background colors independently lets each cell encode two pixel rows.

Combined with 24-bit ANSI escape sequences (ESC[38;2;R;G;Bm), cati achieves full true-color rendering with no image protocol required.

▀fg colorbg color
▄bg colorfg color
█fg colorfg color
⎵transparenttransparent

24×14 source pixels → 24×7 terminal cells

usage

Simple by design

Show an image
$ cati photo.png
$ cati photo.jpg
Show a folder
$ cati ~/Pictures/
$ cati -r ~/Pictures/
$ cati --no-header dir/
Animate frames
$ cati --play frames/
$ cati -p --fps 24 frames/
$ cati -p frame_*.png
All flags
--ansi       24-bit true-color (default)
--play  -p   animate in a loop
--fps  N     frames/sec (default 15)
-r           recurse subdirs
--no-header  suppress filenames
install

Get started in seconds

Requires Go 1.21+

# clone and install
$ git clone https://codeberg.org/ubunatic/cati
$ cd cati && make install

# or build locally and try the logo demo
$ make build
$ ./cati --play --fps 4 assets/
