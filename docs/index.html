terminal image viewer

cat for
images.

cati renders PNG and JPEG files in your terminal using Unicode half-block characters and 24-bit ANSI true-color — no external dependencies, no image protocol required.

capabilities

Everything you need to view images in the terminal

No framebuffer, no image protocol, no root access.

▀▄

Half-block rendering

Each terminal cell encodes two vertical pixels using ▀ ▄ █ and 24-bit true-color ANSI, doubling effective vertical resolution.

📹

Video support

Stream videos (mp4, webm, mkv, mov, avi) inside the terminal. Native frame rate detection automatically matches the video's FPS.

🔍

Interactive viewer

Interactive single-image mode (-i) with mouse-wheel zoom, click & drag panning, arrow-key navigation, and copy to clipboard.

🖼️

Image browser

Run -i on directories or multiple paths to launch a grid preview, page navigation, bottom menu, and full-screen detail views.

⟷

Auto-scaling

Detects terminal size via TIOCGWINSZ and scales media down to fit perfectly — content never breaks your layout.

⌨

Standard keys

Support for PgUp/PgDn, mouse clicks, arrow navigation, and a quick q or ESC to quit and clean up the terminal.

technique

Two pixels, one cell.

Unicode block characters divide a terminal cell into a top half and a bottom half. Setting foreground and background colors independently lets each cell encode two pixel rows.

Combined with 24-bit ANSI escape sequences (ESC[38;2;R;G;Bm), cati achieves full true-color rendering with no image protocol required.

usage

Simple by design

Show an image or video
$ cati photo.png
$ cati video.mp4
Interactive mode & browser
$ cati -i photo.png
$ cati -i folder/
$ cati -i file1.png file2.jpg
Animate / Play sequence
$ cati --play frames/
$ cati -p --fps 30 video.webm
$ cati -p frames_*.png
All flags
--ansi            24-bit true-color (default)
--play        -p  animate in a loop
--interactive -i  interactive viewer/browser
--fps         N   frames/sec (0=auto for video)
-r, --recursive   recurse subdirectories
install

Get started in seconds

Requires Go 1.21+

# clone and install
$ git clone https://codeberg.org/ubunatic/cati
$ cd cati && make install

# or build locally and try the logo demo
$ make build
$ ./cati --play --fps 4 assets/
